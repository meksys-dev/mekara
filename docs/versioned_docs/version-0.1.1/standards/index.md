# Standards

Mekara defines six core standards that make projects AI-first:

- [Standard Mekara Project](./project.md) – Project structure and quality guarantees
- [Standard Mekara Documentation](./documentation.md) – Documentation layout and conventions
- [Standard Mekara Workflow](./workflow.md) – Development pipeline using Claude Code
- [Standard Mekara Command](./command.md) – Structure for natural language command scripts
- [Standard Mekara Design Documents](./design-documents.md) – Standard structure for roadmap and refactor plans
- [Standard Mekara Specs](./specs.md) – Precise module specifications for steady-state design
