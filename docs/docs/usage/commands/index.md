# Commands

mekara provides CLI commands for installation and Claude Code integration:

- [mekara install](./install.md) – Install mekara components (hooks, commands, or both)
- [mekara mcp](./mcp.md) – Run as MCP server for Claude Code
- [mekara hook](./hooks.md) – Hook handlers for Claude Code integration
